# Yapay Zeka Chatbot

## 💡 Proje Hakkında:  
OpenAI API destekli, kullanıcıların sorularını yanıtlayan bir chatbot.

---

## 🚀 Özellikler

- 🧠 OpenAI API ile akıllı, doğal dil yanıtları
- 🌐 Web arayüzü ile kullanıcı dostu deneyim

---

## 🛠️ Kullanılan Teknolojiler

  
- HTML / CSS
- JavaScript
- Node.js  
- OpenAI API
